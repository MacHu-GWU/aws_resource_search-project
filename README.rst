Install
------------------------------------------------------------------------------

``aws_resource_search`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install aws_resource_search

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws_resource_search
